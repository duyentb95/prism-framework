> Định nghĩa thuật ngữ riêng của dự án.

## Thuật ngữ dự án

| Term | Definition | Context |
|------|-----------|---------|
| `regime` | Trạng thái thị trường hiện tại (easy/hard) | Trading strategy |
| `staircase` | Pattern giá grinding từ từ (không spike) | Momentum signal |
| `GSD` | Get Shit Done — làm ngay nếu task < 15 min | Workflow |
| `DoD` | Definition of Done — tiêu chí hoàn thành task | Task management |
| `R` | Risk unit — khoảng cách entry→SL = 1R | Position sizing |
| `Brief for Master` | Báo cáo tóm tắt từ sub-agent → master | Handover protocol |
